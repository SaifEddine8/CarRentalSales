# Car Rental & Sales Platform

A web platform for car rental and sales built with HTML, CSS, and JavaScript. 
This project allows users to register and log in, add cars for sale or rent, and browse available cars. 
Each car listing includes type, price, image, and contact information of the seller.

## Features
- User registration and login
- Dashboard with multiple sections
- Add cars for sale or rent
- Browse cars available for sale and rent
- View car details including image and contact info
- Responsive and styled pages with user-friendly navigation

## Technologies Used
- HTML
- CSS
- JavaScript
